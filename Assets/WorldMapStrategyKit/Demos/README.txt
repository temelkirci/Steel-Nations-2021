Demo Folders
------------

Welcome! In this folder you will find lot of useful demos with sample code.

They are organized in the following categories:

- 2D Map Examples: contains demos of World Map Strategy Kit using the basic 2D projection (flat map)
- General Examples: general demos, usually API related, which can be applied to any view mode (2D, Viewport, ...)
- Path Finding Examples: all path-finding related demos
- Resources: this folder contains materials, textures and models used in the examples
- UI Examples: demos that create some kind of user interface elements
- Viewport Examples: all demos that make use of the viewport projection (3D)
- Terrain: all demos showing how to use WMSK along with Unity standard terrain

Remember to remove the Demo folders when you no longer need them!
